# CommonPopupWindow
Popular popupwindow in the market and customized.

（结合市面流行的弹窗样式+支持自定义布局）. 
# Description
>poplibrary引入即可调用弹窗

>全局配置是否允许重复弹出

>(其他配置待续)

# Feature  

>支持自定义布局(所有点击事件统一回调)  

>仿今日头条更新弹窗
